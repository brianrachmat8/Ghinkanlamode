UPDATE FINAL IMAGE FIT - GHINKAN LA MODE

Perubahan:
1. Semua foto produk tampil utuh/full tanpa kepotong.
2. Berlaku untuk gambar landscape, square, dan portrait seperti foto gamis.
3. Card produk dibuat lebih tinggi agar produk fashion terlihat jelas.
4. Detail produk memakai object-fit: contain.
5. Saat gambar diklik, preview besar langsung menampilkan gambar full tanpa perlu zoom.
6. Background preview memakai gambar yang sama sebagai blur/samar agar tetap premium.
7. Scroll zoom masih tersedia sebagai fitur tambahan, tapi gambar default sudah full.

Cara pakai:
- Extract ZIP.
- Masukkan foto produk ke images/products/.
- Pastikan nama file di script.js sama dengan nama file di folder.
- Buka index.html.
- Tekan CTRL + F5 jika browser masih menampilkan cache lama.
